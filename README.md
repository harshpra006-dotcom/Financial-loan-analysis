# Financial-loan-analysis
Developed a financial loan analysis dashboard using SQL and Excel to track KPIs such as loan applications, funded amount, received amount, average interest rate, and debt-income ratio.
# Objective 
To analyze loan portfolio performance and generate actionable business insights using SQL and Excel dashboards. The project focuses on tracking key loan metrics such as total loan applications, funded amount, total amount received, average interest rate, and debt-to-income ratio (DTI). It also aims to evaluate good and bad loan performance, monitor monthly trends, and support data-driven decision-making through interactive visual reports.
# Key Performance Indicators (KPIs)
1. Total Loan Applications

Measures the total number of loan applications received during a specific period.

2. Total Funded Amount

Represents the total amount of loans disbursed to borrowers.

3. Total Amount Received

Tracks the total repayment amount received from customers.

4. Average Interest Rate

Calculates the average interest rate across all loans.

5. Average Debt-to-Income Ratio (DTI)

Measures borrowers’ financial health by comparing debt obligations to income.

6. Good Loan Percentage

Shows the percentage of loans classified as good loans.

7. Bad Loan Percentage

Shows the percentage of loans classified as bad loans.

8. Good Loan Applications

Total number of approved or fully paid loan applications.

9. Bad Loan Applications

Total number of charged-off or defaulted loan applications.

10. Good Loan Funded Amount

Total funded amount associated with good loans.

11. Bad Loan Funded Amount

Total funded amount associated with bad loans.

12. Good Loan Amount Received

Total repayment amount received from good loans.

13. Bad Loan Amount Received

Total repayment amount received from bad loans.

14. Loan Status Analysis

Analyzes loans based on statuses such as Fully Paid, Current, and Charged Off.

15. Month-to-Date (MTD) Analysis

Tracks monthly performance for applications, funded amounts, and repayments.

16. Month-over-Month (MoM) Analysis

Compares monthly KPI growth and performance trends

# Dashboard Statement

The Financial Loan Analysis Dashboard provides insights into loan performance using SQL and Excel. It tracks key KPIs such as loan applications, funded amount, repayments, interest rates, and DTI to support better business decision-making.

# Tool used
SQL — Data Extraction & KPI Calculation
Excel — Dashboard Creation & Data Visualization

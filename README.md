# Financial-loan-analysis
Developed a financial loan analysis dashboard using SQL and Excel to track KPIs such as loan applications, funded amount, received amount, average interest rate, and debt-income ratio.
# Objective 
To analyze loan portfolio performance and generate actionable business insights using SQL and Excel dashboards. The project focuses on tracking key loan metrics such as total loan applications, funded amount, total amount received, average interest rate, and debt-to-income ratio (DTI). It also aims to evaluate good and bad loan performance, monitor monthly trends, and support data-driven decision-making through interactive visual reports.
